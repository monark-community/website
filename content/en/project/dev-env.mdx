## Developer Environment

- Web hosting platform ([Vercel](https://www.vercel.com/))  
- Web interface using [Next.js](https://nextjs.org/), [Typescript](https://www.typescriptlang.org/), [shadcn-ui](https://ui.shadcn.com/), [tailwindcss](https://tailwindcss.com/)  
- Database and backend hosting([ClickHouse](https://clickhouse.com/),[PostgreSQL](https://www.postgresql.org/) or [GraphQL](https://graphql.org/)) with backend in [Typescript](https://www.typescriptlang.org/) and [Express](https://expressjs.com/)  
- [The Graph](https://thegraph.com/) for blockchain data querying  
- Smart contract library (EVM-based, Solidity, OpenZeppelin)  
- [Docker](https://www.docker.com/) for the containerized environment  
- API documentation using [Swagger](https://swagger.io/) and [OpenAPI](https://www.openapis.org/)
