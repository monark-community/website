## Required Resources

Tools/softwares that are not in the possession of the students will be provided by our team.

- [GIT](https://git-scm.com/)
- [Node.js](https://nodejs.org/) version 22+
- [Visual Studio Code IDE](https://code.visualstudio.com/) and [Remix](https://remix-project.org/) if needed
- Web hosting platform ([Vercel](https://vercel.com/) or [Render](https://render.com/))
- Access to RPC provider
- [Tenderly](https://tenderly.co/) access to create virtual blockchain testnet
- Database ([ClickHouse](https://clickhouse.com/), [PostgreSQL](https://www.postgresql.org/) or [GraphQL](https://graphql.org/)) and Backend ([Express](https://expressjs.com/)) hosting ([Render](https://render.com/))
- Smart Contract Framework ([Solidity](https://soliditylang.org/)) and libraries for secure contract development ([OpenZeppelin](https://www.openzeppelin.com/))
- Sample cooperative data for simulation
