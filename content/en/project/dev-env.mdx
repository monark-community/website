## Developer Environment

- Web interface using **[Next.js](https://nextjs.org/)**, **[Typescript](https://www.typescriptlang.org/)**, **[shadcn-ui](https://ui.shadcn.com/)**, and **[tailwindcss](https://tailwindcss.com/)**
- Database and backend hosting (**[ClickHouse](https://clickhouse.com/)**, **[PostgreSQL](https://www.postgresql.org/)** or **[GraphQL](https://graphql.org/)**) with a backend in **[Typescript](https://www.typescriptlang.org/)** and **[Express](https://expressjs.com/)**
- Smart contract library (**[Solidity](https://soliditylang.org/)** + **[OpenZeppelin](https://www.openzeppelin.com/)**, **[foundry](https://getfoundry.sh/)** and **[Hardhat](https://hardhat.org/)** if needed)
- **[Docker](https://www.docker.com/)** for the containerized environment
- API documentation using **[Swagger](https://swagger.io/)** and **[OpenAPI](https://swagger.io/specification/)**
