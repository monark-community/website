## Description générale

[TimeVault](https://timevault.monark.io) introduit un modèle de contrat intelligent pratique utilisé à travers la finance décentralisée et les opérations DAO : le déblocage de fonds ou de privilèges conditionnel, basé sur le temps. Le cœur de ce système est un contrat intelligent sécurisé qui détient un actif numérique, généralement un token, et ne le libère qu'après un délai spécifié ou à un horodatage de bloc pré-défini.

Les cas d'utilisation courants comprennent le vesting de tokens pour les contributeurs, les paiements d'abonnement, les récompenses différées pour les primes, ou les testaments et les héritages en crypto. Par exemple, un contributeur à une DAO peut se voir attribuer un calendrier de vesting de trois mois, après quoi les tokens se débloquent progressivement. De la même manière, un donateur peut envoyer des tokens à un contrat qui les retient jusqu'à ce qu'un phase caritatif ou une date future soit atteinte.

TimeVault peut être étendu pour inclure des fonctionnalités telles que les déverrouillages récurrents (par exemple, la paie mensuelle), les déverrouillages multi-approbation (par exemple, les systèmes d'escrow) ou l'intégration avec des déclencheurs off-chain comme les oracles pour les conditions réelles (par exemple, l'achèvement d'un projet).

Les étudiants qui construiront ce projet travailleront avec des concepts de contrat intelligent tels que le suivi du temps, la gestion de l'état, la vérification des conditions et le transfert de valeur. Sur le front-end, ils construiront des interfaces utilisateur pour la planification, le suivi et le déverrouillage. TimeVault enseigne comment programmer des règles dans les contrats, garantir l'équité dans le temps et structurer des engagements sans confiance, des compétences essentielles à de nombreuses applications blockchain dans le monde réel.
