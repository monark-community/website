## Environnement de développement

- Plateforme d'hébergement web ([Vercel](https://www.vercel.com/))  
- Interface web utilisant [Next.js](https://nextjs.org/), [Typescript](https://www.typescriptlang.org/), [shadcn-ui](https://ui.shadcn.com/), [tailwindcss](https://tailwindcss.com/)  
- Hébergement de base de données et backend([ClickHouse](https://clickhouse.com/), [PostgreSQL](https://www.postgresql.org/) ou [GraphQL](https://graphql.org/)) avec backend en [Typescript](https://www.typescriptlang.org/) et [Express](https://expressjs.com/)  
- [The Graph](https://thegraph.com/) pour l'interrogation des données blockchain  
- Bibliothèque de contrats intelligents (basée sur [EVM](https://ethereum.org/developers/docs/evm/), [Solidity](https://soliditylang.org/), [OpenZeppelin](https://www.openzeppelin.com/))
- [Docker](https://www.docker.com/) pour l'environnement conteneurisé  
- Documentation API utilisant [Swagger](https://swagger.io/) et [OpenAPI](https://www.openapis.org/)
