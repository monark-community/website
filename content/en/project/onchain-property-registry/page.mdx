## General Description

[Cadastrum](https://cadastrum.monark.io) is a civic infrastructure project that leverages blockchain to store and maintain a public registry of land and building data. Users can submit updates to parcel-level information such as construction year, structural modifications, zoning details, and other compliance-relevant attributes. Each update is tied to a wallet address, timestamped, and stored immutably on-chain, creating a clear audit trail for inspectors, municipalities, and property owners.
        
This registry addresses current challenges in outdated, inaccessible, or fragmented land records. It allows communities to collaboratively track development activity, verify structural changes, and flag irregularities. Through a map-based interface, users can view the history and current status of any lot, submit new updates, and (optionally) attach documentation or media files via [IPFS](https://ipfs.tech/).
        
Advanced features may include role-based submissions (e.g., homeowner vs. city official), voting or validation for changes, and cross-integration with zoning or land use proposals. Students will build on-chain storage systems, map overlays, and frontend tools for contributing and browsing property data. This project is ideal for those interested in civic tech, smart cities, and decentralized public services.
